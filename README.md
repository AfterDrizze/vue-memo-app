\# Vue 3 智能备忘录应用



一个基于Vue 3的现代化备忘录/待办事项管理应用。



\## ✨ 功能特性

\- ✅ 用户登录/注册系统

\- ✅ 文件夹分类管理

\- ✅ 备忘录的增删改查

\- ✅ 内容实时编辑与自动保存

\- ✅ 响应式布局设计

\- ✅ 数据本地持久化存储



\## 🛠 技术栈

\- Vue 3 (Composition API)

\- Vue Router 4

\- Pinia (状态管理)

\- Element Plus (UI组件库)

\- Vite (构建工具)
